# libjam
A Python library that makes it easier to create better CLIs and provides some missing pieces for file management.

## Installation
libjam is available on [PyPI](https://pypi.org/project/libjam), and can be installed using pip.
```sh
pip install libjam
```

## Documentation
The documentation, with examples, is available on [here](https://libjam.readthedocs.io).
